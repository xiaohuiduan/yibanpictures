## 易班图片网站

网站主要是分为两个部分：

- 斗图网站

- 图片颜值查询

### 斗图网站

斗图网站是接入斗图啦网站的API，[地址](https://www.doutula.com/)

网页后台使用的是maven构建项目，springmvc框架。前端由吴逸尘所写，使用的是jsp构建前段页面。
